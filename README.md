Here we collect SHEMAT-Suite input files including results. These may serve as a
benchmark test in the future. 


Naming convention:

runmode_props-module_Description


Examples: 

fw_bas_2D-Heat-Transfer\

fw_mphase_const_2D-CO2-injection

sm_bas_MonteCarlo


Every Model folder should contain a /doc folder, which describes how to compile 
and run the model. And a /results folder which collects the output results. 
A short description of the model can also be put in the /doc folder.
